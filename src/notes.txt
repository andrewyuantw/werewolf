Server messages

Client to Server

JOIN_GAME - whenever a new person submits their display name 
START_GAME - whenever the host presses the START GAME button
READY_TO_START - Once they see the character role, players hit ready

Server to Client

JOIN_LOBBY - whenever a new person submits their display name (every person's lobby list needs to now be updated)
ENOUGH_PEOPLE - whenever we reach enough people in the lobby for a game to start, we send this message to the host
START_GAME - tells every player that the game has now started
GO_TO_NIGHT - goes into night
